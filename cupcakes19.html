PROBLEM 1 (30 points): First Knight

Given an nxn chessboard, you're tasked with finding the minimum number of movees the knight would require to reach a given sqaure (x,y) from the bottom-left square (1,1). Knight moves 2 squares in one direction and 1 square in another. Here's a demonstration. (NOTE: There aren't any other pieces on the board)

Input Format: The first line consists of 3
space-separated integers n, x and y where n represents
the dimension of the board(nxn), x represents the
column (1 being the left-most) and y represents the row
(1 being the lower-most).
Output Format: One integer denoting the minimum
number of moves required to reach given destination.


Sample Input
8 3 2


Output
1
