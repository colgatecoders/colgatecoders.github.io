Utilize your knowledge of Python to make beautiful patterns.

Setup Instructions

Option 1. (RECOMMENDED)

1. Open terminal (or command-prompt or bash).
2. Run "pip install turtle" Install package turtle
3. Install jupyter by using "pip install jupyter"
4. Download problems.zip
5. Unzip level100.zip.
6. Run "cd level100 && jupyter notebook Cupcakes100.ipynb".
6. Run "cd level100;jupyter notebook Cupcakes100.ipynb".

Option 2

. If you run into issues with installing jupyter or turtle

1. View the problems here
2. Use online IDE at trinket and solve the problems.
